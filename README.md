# The Kitchen Shelf

A responsive e-commerce app built with React, React Router, React Bootstrap, and DummyJSON.

## Features

- Landing page with store introduction
- Products page showing kitchen-related products from DummyJSON
- Product details page with image, price, rating, description, and reviews
- Add to cart functionality
- Cart page with quantity summary and remove-item option
- Favourites page for saved products
- Payment placeholder page
- Responsive layout for mobile and desktop screens

## Tech Used

- React
- React Router
- React Bootstrap
- CSS
- DummyJSON API

## What I Practiced

- Fetching data from an API
- Passing props between components
- Using React state for cart and favourites
- Creating dynamic routes with React Router
- Building reusable components
- Styling responsive product cards
